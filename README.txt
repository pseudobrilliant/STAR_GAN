The included code contains a python implamantion of STARGAN Based off a previous ACGAN model. To run the code in a linux environment please install the prerequistes using the provided setup file or use pip and the requirements file. If any libraries are missing please contact me, so that I may make the updates.

To run the code please activate the pyenv venv (GANEnv on windows) created by the setup script source ./venv/bin/activate python main.py

Please be aware that running this code will download a large number of files for training. This will take some time.
